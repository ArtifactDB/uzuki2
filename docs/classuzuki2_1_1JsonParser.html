Detailed Description

Parse JSON file contents using the uzuki2 specification.

JSON provides an alternative to the HDF5 format that is handled by Hdf5Parser. JSON is simpler to parse and has less formatting-related overhead. However, it does not support random access and discards some precision for floating-point numbers.
