◆ Format

enum uzuki2::StringVector::Format

Format constraints to apply to the strings.

  • NONE: no constraints.
  • DATE: strings should be in YYYY-MM-DD format.
  • DATETIME: strings should be in Internet Date/Time format (see RFC 3339, section 5.6).
